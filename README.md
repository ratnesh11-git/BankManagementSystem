# Bank Management System (C++)

A console-based Bank Management System developed using C++. This project allows users to create accounts, deposit and withdraw money, and view account details. It demonstrates the use of Object-Oriented Programming (OOP) concepts and file handling.

## 🚀 Features
- Create a new bank account
- Display all account details
- Deposit money into an account
- Withdraw money from an account
- Data stored using file handling

## 🛠️ Technologies Used
- C++
- File Handling
- Object-Oriented Programming (OOP)

## ▶️ How to Run

1. Compile the program:
   g++ bank.cpp -o bank
2. To Run The Program:    
   ./bank
